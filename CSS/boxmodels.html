Box Model

The CSS box model describes the rectangular boxes generated for elements in the document tree and the layout of these boxes. It consists of the following components:

  • Content: The innermost part of the box where text and images appear.
  • Padding: The space between the content and the border. It can be set using the padding property.
  • Border: The area surrounding the padding (if any) and content. It can be styled using the border property.
  • Margin: The outermost layer that creates space between the element and other elements. It can be set using the margin property.
