LISTS IN CSS

CSS provides various properties to style lists:

  • list-style-type: Specifies the type of list item marker (e.g., disc, circle, square).
  • list-style-position: Specifies the position of the list item marker (e.g., inside, outside).
  • list-style-image: Specifies an image to use as the list item marker.
