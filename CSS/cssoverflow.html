CSS OVER FLOW

CSS overflow property specifies what happens if content overflows an element's box. It can take the following values:

  • visible: The overflow is not clipped. It renders outside the element's box. This is the default value.
  • hidden: The overflow is clipped, and the rest of the content will be invisible.
  • scroll: The overflow is clipped, but a scroll-bar is added to see the rest of the content.
  • auto: If overflow is clipped, a scroll-bar should be added to see the rest of the content.
